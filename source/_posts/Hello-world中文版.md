## 快速启动

### 创建一个新帖子

``` bash
$ hexo new "新帖子"
```

更多信息[写作](https://hexo.io/docs/writing.html)

### 运行服务器

``` bash
$ hexo server
```

更多信息[服务器](https://hexo.io/docs/server.html)

### 生成静态文件

``` bash
$ hexo generate
```

更多信息:[产生](https://hexo.io/docs/generating.html)

### 部署到远程站点

``` bash
$ hexo deploy
```

更多信息[部署](https://hexo.io/docs/one-command-deployment.html)
